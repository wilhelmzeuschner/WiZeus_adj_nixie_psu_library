Author
Wilhelm Zeuschner
Date
03. October 2021
Version
1

This class implements the controls for the HV PSU

There are two ways to initialize an instance of this class The default constructor takes no parameters and assumes the default values for the feedback resistors If these values do not match the hardware configuration, the second constructor option allows to pass custom values

int adjNixiePSU::setOutputVoltage ( unsigned int  outputVoltage)

Sets the output voltage to the specified value.

Parameters
outputVoltageDesired output voltage
Returns
Voltage value that was set. Since the passed parameter might exceed the min and max allowed voltages. Returns -1 if no connection to the digi pot could be established
Attention
Since there is NO feedback of the actual output voltage to the Arduino, the actual output voltage might vary. The module itself does use a closed loop control system. This means that the module will attempt to maintain the voltage, independent of the load (within its physical limitations).
